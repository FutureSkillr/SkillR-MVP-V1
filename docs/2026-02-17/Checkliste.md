# Checkliste: Wann war unsere MVP-Session erfolgreich?

*Basierend auf der Diskussion vom 17.02.2026*

---

## VUCA-Reise funktioniert

- [ ] Kann ein Jugendlicher die App starten und versteht intuitiv, was zu tun ist?
- [ ] Funktioniert der Einstieg über ein frei gewähltes Interesse (z.B. "Holz", "Kochen", "Holocaust")?
- [ ] Generiert das LLM sinnvolle, altersgerechte Inhalte dynamisch im Dialog?
- [ ] Werden Orte vorgeschlagen, die thematisch zum Interesse passen?
- [ ] Hat der Nutzer bei jedem Schritt die Wahl: lokal oder global, Thema vertiefen oder wechseln?
- [ ] Entsteht kein Bias (nicht nur noch Holz, wenn einmal Holz gewählt wurde)?
- [ ] Werden alle vier VUCA-Dimensionen über die Reise hinweg abgedeckt?
- [ ] Gibt es ein erkennbares Abschlusskriterium ("Bingo-Logik")?

## Skill-/Interessenprofil entsteht

- [ ] Wird aus den Dialogverläufen und Antworten ein sichtbares Profil aufgebaut?
- [ ] Verändert sich das Profil-Diagramm (Spinnennetz) erkennbar nach jeder Lektion?
- [ ] Kann der Nutzer sein Profil am Ende einsehen und nachvollziehen?
- [ ] Speichert das System die Daten so, dass beim nächsten Besuch daran angeknüpft wird?

## KI-Coach / Dialogqualität

- [ ] Funktioniert die Sprachausgabe und ist die Stimme angenehm?
- [ ] Reagiert der Coach sinnvoll auf Antworten (kein generischer Einheitsbrei)?
- [ ] Kann man sowohl per Text als auch per Sprache interagieren?
- [ ] Stellt das System Verständnisfragen, die Soft Skills und Future Skills adressieren?

## Zielgruppen-Akzeptanz

- [ ] Sagt ein 14-Jähriger nach dem Test: "Das war nicht wie eine Lern-App"?
- [ ] Würde ein IHK-Vertreter nach der Demo verstehen, welchen Mehrwert das Interessenprofil hat?
- [ ] Können Eltern nachvollziehen, was ihr Kind in der App gemacht hat?
- [ ] Sieht ein Unternehmer, wie perspektivisch ein Matching auf seine Bedarfe entstehen könnte?

## Technisch stabil

- [ ] Läuft die Web-App im Browser ohne Installation?
- [ ] Sind die Antwortzeiten akzeptabel (kein langes Warten auf LLM-Generierung)?
- [ ] Werden die Profildaten zuverlässig gespeichert (Backend/Datenbank)?
- [ ] Kostet der Betrieb bei den ersten ~100 Testnutzern noch im freien Google-Kontingent?

## Abgrenzung eingehalten

- [ ] Spricht das System keine Berufsempfehlungen aus?
- [ ] Gibt es keine vorgefertigten Kurskataloge?
- [ ] Wird nirgendwo das Wort "Zertifikat" verwendet?
- [ ] Sind keine Funktionen eingebaut, die wir bewusst ausgeklammert haben?

---

> **Der ultimative Erfolgstest:** Ein Jugendlicher macht die Reise freiwillig zu Ende, schaut auf sein Profil und sagt: "Stimmt, das bin ich."
