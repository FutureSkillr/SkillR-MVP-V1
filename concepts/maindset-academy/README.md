## Three-Entity Model

| Entity | Type | Role |
|--------|------|------|
| **SkillR** | Kids Education Brand | Kids education brand & sub-branding provider — app on top of maindfull.LEARNING by maindset.ACADEMY. Solid Pod-based lifelong profiles. Learning-sponsoring module (resellable). B2C + B2B |
| **maindset.ACADEMY** | New Learning Brand (owns engine) | New learning education brand — owns and operates maindfull.LEARNING. Lernreisen, skill validation, skill signatures, dedicated frontend, content. B2C + B2B |
| **maindfull.LEARNING** | AI Engine (by maindset.ACADEMY) | New learning AI engine by maindset.ACADEMY. Powers smart courses and adaptive learning |
